# My Professional Portfolio

A modern, minimalist portfolio website built with HTML, CSS, and hosted on GitHub Pages.

## Features

- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- 🎨 **Modern Design** - Clean and professional minimalist aesthetic
- ⚡ **Fast Loading** - Pure HTML/CSS with no external dependencies
- 🚀 **GitHub Pages Ready** - Deploy directly from your repository
- ♿ **Accessible** - Semantic HTML and WCAG compliant

## Sections

- **Home** - Eye-catching hero section with call-to-action
- **About** - Your bio and skills
- **Projects** - Showcase your best work with descriptions
- **Experience** - Professional background and achievements
- **Contact** - Easy ways for visitors to reach you

## Customization Guide

### 1. Update Your Information

Edit `index.html` and replace:
- `Your Name` - Your actual name
- Personal bio in the About section
- Project details and links
- Experience history
- Contact information

### 2. Personalize Colors

Edit `styles.css` and modify the CSS variables:

```css
:root {
    --primary-color: #1a1a1a;
    --secondary-color: #333333;
    --accent-color: #0066cc;
    --text-light: #666666;
    --bg-light: #f5f5f5;
}
